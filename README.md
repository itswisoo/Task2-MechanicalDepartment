# Chess Pawn Design

## Overview
This repository contains a 3D chess pawn model created using Onshape as part of my training tasks. The model was designed from a reference image and exported as an STL file.

## Workflow
1. Selected a chess pawn as the reference shape.
2. Created a new document and started a sketch on the Front plane.
3. Drew the pawn using Line and Structural Curvature tools.
4. Applied the required dimensions to complete the sketch.
5. Added my first initial and my father's first initial at the bottom of the model.
6. Extruded the sketch to create a 3D model.
7. Exported the final model as an STL file.
